== Description ==
This adds a new button on the media page, which allows to link a media to a post, without inserting it into the content. 
If the link to the post already exists, it will remain the old link.

== Installation ==
= Requirements =
* WordPress version 2.9 and later (tested at 3.4.2)
* PHP 5.2

= Installation =
1. Unpack the download-package
1. Upload the folder and all folder and files includes this to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Thats all

= Usage =
See the tutorials for use the default functions from WordPress to use the images on theme
